# SI-GuidedProject-3409-1624086363
Software requirements:
1.    Anaconda 
2.    Any browser

How to run the app:
 1.   Download zip file of Book-Recommendation-main folder.
 2.   Extract all the files from the zip file.
 3.   Now open Anaconda prompt and set the directory path upto templates folder.
 4.   Enter the command "python app.py" and click enter.
 5.   Copy the URL which you will get in the Anaconda prompt and paste it in any browser.
